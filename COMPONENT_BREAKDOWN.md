# FreshMart Website Component Breakdown

The original FreshMart website can be separated into these six components:

1. **User Authentication**
   - Register
   - Login
   - Logout
   - Session handling
   - Profile update
   - Admin user list/search/delete

2. **Product Management**
   - Product list
   - Product details
   - Product search/filter
   - Admin add/edit/delete products
   - Stock and category management

3. **Shopping Cart**
   - Add item to cart
   - View cart
   - Update quantity
   - Remove item
   - Clear cart

4. **Order Management**
   - Place order
   - View order history
   - View order details
   - Cancel order
   - Admin order status updates

5. **Payment & Checkout**
   - Checkout page
   - Delivery/address collection
   - Payment method handling
   - Order confirmation flow

6. **Admin Dashboard**
   - Admin navigation
   - User, product, and order management links
   - Summary cards/statistics
   - Admin-only access control

This ZIP contains only **Component 1: User Authentication**.
